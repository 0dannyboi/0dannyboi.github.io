The Logistic Map

Please check out the Python code I used to study the map here.
The prototypical example of a discrete dynamical system is the logistic map, which describes the dynamics of a population in terms of a reproduction rate, \(r\). At any particular iteration, \(n\), we can define the ratio of the population to the systemâ€™s carrying capacity as \(x_n\), a value between 0 and 1.

\[ x_{n+1} = r * x_n (1 - x_n)\]

Intuitively, the reproduction rate tends to increase the population. However, the model accounts for resource scarcity through the factor \((1 - x_n)\), which decreases linearly with the population. As a result, the population tends to increase linearly as \(x_{n+1} \approx r * x_n \) for small populations. As the population approaches the carrying capacity ( \( x_n \sim 1 \) ), the resource scarcity factor approaches 0, driving the population towards 0.

Orbits and Cobweb Plots

We can describe the trajectory that the system takes from some initial condition \(x_0 \) by looking at the set of values the system covers. This set \( \{x_0, x_1, x_2, ... \} \) is a particular orbit.

For 1-dimensional maps, such as the logistic map it can be easy to determine if a particular orbit diverges, converges towards a particular value (or cycle of values), or does something else by tracing the orbits under iteration. The cobweb plot is a common visual representation for the orbits of 1-dimensional systems. The basic structure of the algorithm is as follows:

\(1\). Plot the line \(y = x\) and the map \(f\) as \(y = f(x)\).

\(2\). Note the fixed points found where the line \(y = x\) intersects \(f\). The values of these fixed points are invariant under the map \(f\). An orbit that arrives here will not leave.

\(3\). Draw a vertical line from the initial condition to the curve \(y=f(x)\). This line starts at \( \left(x_0, 0 \right) \) and leads to the point \( \left(x_0, f(x_0) \right) \).

\(4\). Draw a horizontal line from the point on \(f(x)\) from step 3 to the line \(y = x\). This line starts at \( \left(x_0, f(x_0) \right) \) and leads to \( \left(f(x_0), f(x_0) \right) \)

\(5\). Draw a vertical line from the value \(\left(f(x), f(x)\right)\) on the line \(y = x \) towards the map \(f\). The point we land on here is \( \left(f(x), f(f(x))\right) \).

\(6\). Repeat on from step 4.

Continuous Dynamical Systems

A simple continuous dynamical system can be realized by considering a pendulum consisting of a mass, \( m \), suspended on a massless rod of length \(L \). The equations of motion can be obtained by computing the Lagrangian, \(\mathcal{L} = T - U\) and evaluating the Euler-Lagrange equation: \[ \frac{d }{d t} \frac{\partial \mathcal{L} }{\partial \dot{\theta}} = \frac{\partial \mathcal{L}}{\partial \theta} \] Where the kinetic energy of the system is: \[ T = \frac{m L^2}{2} \dot{\theta}^2 \] and the potential energy of the system is: \[ U = m g L (1 - \cos(\theta)) \] The equation of motion of the system becomes: \[ \ddot{\theta} = - \frac{g}{L} \sin(\theta) \] We can rewrite this second order differential equation as a coupled set of first order ODEs: \[\frac{d \theta}{d t} = \dot{\theta} \] \[\frac{d \dot{\theta}}{d t} = \frac{g}{L} \sin(\theta) \] Further pages show how this representation allows us to visually represent the dynamics of this system.
